# Day 2 - What is Prompt Engineering?

## What I Learned Today
Prompt Engineering is the skill of writing clear instructions to get the best response from AI like Claude.

## Lazy Prompt
**Prompt used:** "Explain Prompt Engineering"
**Result:** Basic explanation, no structure

## Engineered Prompt
**Prompt used:** ABTalks Prompt Template
**Result:** Structured response with sections and examples

## Key Learnings
- Specific prompts give better results
- Vague prompts = vague answers
- Engineered prompts save time

## Weak vs Improved Prompt
- Weak: "Write about marketing"
- Improved: "Write a 200-word LinkedIn post for small business owners"
