# 🥗 NutriScope - AI Nutrition Analytics App

## 📌 Overview

NutriScope is an AI-powered Nutrition Analytics application built to help users track food intake, analyze nutrient consumption, identify deficiencies, and receive personalized dietary recommendations.

This project was developed as part of the Day 9 AI Development Challenge, focusing on iterative application development using AI-generated prompts.

---

## 🚀 Version 1 (MVP)

### Features

- User Profile Management
  - Age
  - Gender
  - Height
  - Weight
  - Activity Level
  - Dietary Preference

- Food Logging System
  - Add Food
  - Quantity & Unit Selection
  - Editable Entries
  - Remove Food Items

- Nutrition Tracking
  - Calories
  - Protein
  - Carbohydrates
  - Fat
  - Fiber
  - Iron
  - Calcium
  - Vitamin C
  - Vitamin D
  - Vitamin B12

- Dashboard
  - Energy Progress
  - Macro Distribution
  - Nutrient Tracking
  - Deficiency Detection
  - Excess Detection

- Smart Recommendations
  - Food Additions
  - Food Swaps
  - Portion Adjustments
