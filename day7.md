# Day 7 - Claude Model Selection & Reasoning Effort

## My Claude Usage Strategy
- **Use Case:** Learning & Research
- **Plan:** Free

## Model Selection Guide

| Task Type | Best Model | Why |
|---|---|---|
| Quick questions | Haiku | Fast, saves limits |
| Summarizing articles | Sonnet | Balanced depth |
| Complex analysis | Sonnet | Best for free tier |
| Deep reasoning | Sonnet (extended thinking) | Worth saving limits |

## Reasoning Effort Strategy
- **Low effort** → Quick facts, definitions
- **Medium effort** → Explaining concepts, comparing ideas
- **High effort** → Complex problems, research synthesis

## Free Plan Tips
1. Batch multiple questions into one message
2. Be specific upfront to avoid back-and-forth
3. Save limits for deep/complex tasks
4. Use Projects to store context
5. Do heavy work early (daily limit resets)

## Sample Research Workflow
1. Paste source + ask for structured summary
2. Ask Claude to find gaps or counterarguments
3. Request study outline or notes template
4. Final message to synthesize everything

## Key Learning
Precision beats frequency on free tier.
One well-crafted prompt beats five vague ones.
